## üöÄ Installation

### Windows Terminal

1. Open Windows Terminal
2. Press `Ctrl + ,` to open Settings
3. Click "Open JSON file" (bottom left)
4. Add this theme to the `"schemes"` array:
   ```json
   {
     "name": "Eclipse Midnight",
     "background": "#0f0e15",
     "foreground": "#e2e2e2",
